Попытка частичной автоматизации курсовой по логическому проектированию.

В текущей реализации работает только в Visual Studio, основная проблема - 8битная кодировка типа char.

Большая часть отчета генерируется в tex, но есть методы для генерации csv таблиц
Пример tex отчета: stateMachine.tex

Для создания изображений кубов необходим генератор nop из пакета graph
пример использования из консоли dot -Knop -Tpdf cube.dot -o cube.pdf

Файл stateMachineFormulasExample.txt содержит пример задания автоматных уравнений по кубам
Он делается пользователем и на его основе генерируется кусок отчета с изображениями кубов и уравнениями
Для этого файлы с изображениями кубов должны иметь то же имя что и .dot файл и расширение .pdf
